# 我的记账系统终于能电脑手机同步了

这次我做了一个自己的记账系统，名字叫：

```text
闫的记账铺
```

它可以记收入、支出、分类、账户，还能看图表。

最重要的是：电脑和手机都能打开同一个网址。

![记账系统首页截图](/image/20.png)

## 最后使用的网址

真正打开记账系统的网址是：

```text
https://jizhang-demo.pages.dev
```

手机和电脑都打开这个。

如果页面没变化，可以打开：

```text
https://jizhang-demo.pages.dev/?v=pages
```

## 一开始遇到的问题

刚开始我用的是电脑本地地址，类似：

```text
http://LAPTOP-L91VHNS3:5178
```

电脑能打开，但是手机打不开。

后来又试了一个同步地址：

```text
https://yan-jizhang-sync.495662876.workers.dev
```

电脑能用，但是手机打不开，所以手机上传账单会提示：

```text
上传失败：Load failed
```

![手机上传失败截图](/image/21.png)

最后改成了更简单的办法：  
网页和同步都用同一个地址。

```text
https://jizhang-demo.pages.dev
```

这样手机不用再访问那个打不开的 `workers.dev` 地址。

## 怎么同步

打开记账系统后，进入：

```text
设置 -> 云同步
```

同步接口地址填写：

```text
https://jizhang-demo.pages.dev
```

然后点击：

```text
保存同步设置
```

![云同步设置截图](/image/22.png)

## 电脑和手机的同步节点

我现在理解的同步过程很简单：

```text
电脑 / 手机
   ↓
闫的记账铺网页
   ↓
云端账本
   ↓
另一台设备再同步回来
```

也就是说，电脑和手机不是直接互相传数据，而是都连到同一个云端账本。

我准备在这里放一张图，展示电脑端和手机端同步的位置：

![电脑端和手机端同步节点](/image/23.png)

## 如果电脑上有账单

电脑上记了账，就在电脑上点：

```text
上传本机到云端
```

然后手机打开同一个网址，登录同一个账号，再点：

```text
从云端同步到本机
```

这样电脑上的账单就到手机了。

## 如果手机上记账

如果平时主要用手机记账，就在手机上这样设置：

```text
设置 -> 云同步
同步接口地址：https://jizhang-demo.pages.dev
勾选：保存账单后自动上传到云端
```

以后手机每次记账，会自动上传。

电脑想看手机上的账单时，打开电脑端，点：

```text
从云端同步到本机
```

简单理解就是：

```text
在哪台设备记账，就在哪台设备上传。
另一台设备想看，就从云端同步到本机。
```

## 记账界面能做什么

记账页面主要就是日常用的功能，不复杂：

- 可以选择消费或收入
- 可以填写金额
- 可以选分类，比如餐饮、交通、购物、工资
- 可以选账户，比如现金、银行卡、支付宝、微信
- 可以写备注
- 保存后会出现在账单列表和统计图表里

## 多个人使用会不会混在一起

不会。

系统不是只靠密码区分用户，而是靠：

```text
用户名 + 密码
```

用户名才是区分账本的关键。

比如两个人密码一样，但是用户名不同，账本也不会混在一起。

```text
张三 / 123456
李四 / 123456
```

这两个账号会是两个独立账本。
